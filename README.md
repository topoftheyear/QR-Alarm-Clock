# QR-Alarm-Clock

An alarm clock app that requires the user to scan a QR code to shut off the alarm.

## Getting Started

Clone and open in Android Studio. Build and run.

## Built With

* Android
* [ZXingScannerView](https://github.com/dm77/barcodescanner)
